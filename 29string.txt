A sliding window is a powerful technique used to solve problems involving subarrays or substrings efficiently â€” especially when the subarray length is fixed or changes predictably.
ğŸ“¦ Core Idea:

Instead of recomputing values from scratch for every subarray/substring, we:

    Process the first window.

    Slide the window one element at a time.

    Update values incrementally (add new element, remove old one).
